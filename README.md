<h2>CALYX</h2>
Calyx: Empowering Safety for Everyone
Calyx is a comprehensive safety application designed to provide immediate assistance during emergencies. While built with women’s safety as a priority, Calyx is a versatile tool for children, the elderly, and anyone navigating potentially unsafe environments or traveling alone.
## Features
🚨 Instant SOS Emergency System
With a single trigger, Calyx initiates a multi-layered response to ensure the user is never alone in a crisis:
• Emergency Calls: Automatically dials pre-configured emergency contacts and the nearest police station.
• Real-time SMS: Sends an immediate distress message to your trusted circle.
• Live Location Tracking: Shares your precise coordinates in real-time so help can find you faster.
🛡️ Anonymous Reporting
We believe in accountability without fear. The Anonymous Report feature allows users to document "bad experiences" or safety concerns:
• Evidence Upload: Attach photos, videos, or audio recordings of the incident.
• Direct Submission: Reports are forwarded to the relevant police authorities while keeping your identity protected.
• Community Safety: Helps authorities identify "hotspots" of harassment or danger.

📍 Travel & Accident Security
Whether it’s a late-night cab ride or an unexpected medical emergency, Calyx acts as a digital guardian, ensuring that your location and status are always accessible to those who care.
## How It Works
1. Registration: Users set up their profile and input "Emergency Contacts" (Family, Friends, or Guardians).It has inbuild access to emergency helplines.
2. Activation: In a moment of danger, the user triggers the SOS.
3. Redirection: For non-immediate threats, the user is redirected to the Anonymous Reporting portal to file a detailed account of an incident.
## Tech Stack
• Frontend: html,css,js
• Backend: json,flask,python
• APIs: Google Maps API (for Live Location), Twilio (for SMS/Calls).
